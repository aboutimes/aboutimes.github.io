Markdown 基本语法

2017-08-06 | 笔记 | Couh

　　Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯（ John Gruber ），在语法上有很大一部分是跟亚伦·斯沃茨（ Aaron Swartz ）共同合作的。这个语言的目的是希望大家使用易读易写的纯文本格式编写文档，然后转换成有效的 XHTML（ 或者 HTML ）文档。这种语言吸收了很多在电子邮件中已有的纯文本标记的特性。
