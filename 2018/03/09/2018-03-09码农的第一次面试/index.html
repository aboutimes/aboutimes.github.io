第一次 PHP 面试

2018-03-09 | 见闻 | Couh

　　早上吃过早餐，忙着恶补代码基础，接到座机来电，一下子紧张起来，会不会是面试电话呢？接起电话听到声音甜美的妹子说到：“你好，是某某某吗，我们收到你的简历，想请您过来面试……” 哇，炸了，差点笑出来，还好我的反应快，收住笑脸忍住笑声接着把话说完。挂断电话，高兴得恨不得向世界宣布，我要去面试啦，好在理智胜过了冲动，一想到自己的基础，说实在，心里是忐忑的，很大可能就是炮灰啊，于是赶紧坐回来看文档。这就是此刻的心情，决定转行当个程序员以来第一回正式投简历，并接到面试通知，纪念一下。
　　中午赶去南宁，下车直接前往公司，公司办公面积虽然不大，不过装饰挺潮的，像个咖啡厅，是喜欢的风格，阳台上有个休闲的小吧台，面试就在吧台这儿，高高的吧台椅有些不习。面试并不顺利，了解了一些 PHP 框架的情况，问了些 Javascript 标签操作的问题，没答上来，本来也觉得 Javascript 比较薄弱，结果就给问到了，果然是着了墨菲定律。之后又问了 Linux 文件执行权限的设置，哎，以前倒是没怎么关注执行权限，平时都是偷懒给全部权限，这下又着了。最后了解了下熟悉的 Javascript 框架，公司打算用 Vue 重构前端，再次不会，完蛋了，结果可想而知，等通知……
　　之后陆续面试了两家，都不太顺利，不过倒是暴露出许多问题，发现许多不足。面试的确是检验学习成果的重要手段，如考试一般，考得满意固然好，考得差反而能暴露出不足，给人予加倍的学习动力，继续努力吧，少年。

面试
